# AI 求职助手 · JobCoach

面向大学生求职场景的 AI 求职工具 —— 产品经理实习作品。

## 功能

| 功能 | 说明 |
|------|------|
| JD 分析 | 单份拆解 + 多份横向对比（找共性/差异/词频） |
| 简历优化 | STAR 改写 + 匹配度分析 + PDF 上传 |
| JD-简历差距分析 | 自动对比简历与 JD 的技能缺口、给出改写清单 |
| 学习路线 | 基于 JD 分析生成按周准备计划（2-8 周） |
| AI 模拟面试 | 50+ 岗位、6 轮动态追问、6 维度评估、进步追踪 |

## 技术

Streamlit + DeepSeek API · Python 3.10+ · pypdf

## 在线体验

`https://ai-career-coach-5jruwnlitbcw7pjsqqoesj.streamlit.app`

## 产品叙事（面试用）

- 用户调研：5 位同学试用反馈驱动迭代
- Prompt 迭代：15+ 版——追问策略从固定维度改为自适应，面试官人设动态注入岗位
- 版本节奏：v1.0→v2.1，6 个迭代，每次有明确决策依据（见侧边栏"关于"）
- 设计原则：灰白极简风，刻意去 AI 味，不做另一个紫蓝渐变产品
