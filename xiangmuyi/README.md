# AI Career Copilot · AI 职业成长助手

从"我不知道该做什么"到"我知道该怎么准备"——陪伴大学生完成职业探索与求职成长的每一步。

## 定位

面向 26-29 届大学生及应届毕业生的 AI 职业成长助手。不是"帮你写简历的工具"，而是"帮你搞清楚自己该往哪走、怎么走"的职业成长伴侣。

## 核心价值

| 阶段 | 功能 | 说明 |
|------|------|------|
| 🧭 职业探索 | AI 引导式问答 | 不知道适合什么岗位？回答 3 个问题，AI 推荐方向 |
| 📋 岗位认知 | JD 分析 + 岗位百科 | 看不懂 JD？AI 拆解岗位要求，告诉你这个岗位日常做什么、需要什么能力 |
| 📊 能力评估 | 简历优化 + 差距分析 | 简历和岗位差在哪？AI 做匹配度分析，给具体补救方案 |
| 🗺️ 学习规划 | 按周学习路线 | 知道差什么但不知道怎么补？AI 生成 2-8 周结构化计划 |
| 💬 求职准备 | AI 模拟面试 | 没人陪练？6 轮动态追问 + 多维度评估 + 结构化反馈 |
| 👤 持续成长 | 成长记录 + 推荐 | 面试进步追踪 + 下一步行动建议 |

## 产品叙事（面试用）

- **版本迭代**：v2.1（原JobCoach）→ v3.0（AI Career Copilot），产品定位从"AI产品经理求职工具"升级为"全职业方向成长助手"
- **用户调研**：5 位同学试用反馈驱动迭代
- **Prompt 迭代**：15+ 版，追问策略从固定维度改为自适应，面试官人设动态注入岗位和 JD
- **设计原则**：灰白极简风，刻意区别于 AI 产品的紫蓝渐变色，降低用户对"AI 味"的抵触
- **核心设计决策**：全流程数据桥接（JD→简历→学习路线→面试），不依赖用户会写 Prompt
- **面试评估机制**：隐式评估（每轮私下记录，最后一轮汇总），避免评估偏见；评估维度按岗位类别自适应
